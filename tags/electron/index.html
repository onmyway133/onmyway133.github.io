How to change app icon in electron

Generate icns

  • Generate .iconset
  • Run iconutil -c icns "Icon.iconset". Note that icon names must be first letter lowsercased, and use _ instead of -

Use icns

  • In main.js, specify icon

You can also use helper url methods
